
1.由于含exe文件的压缩包无法上传到PTA，我们将exe文件的后缀改成了txt，对应的文件是“运行程序.txt”，使用时将后缀修改成exe即可。如果修改后缀无法正常运行的话，就利用所有cpp,h文件重新生成就好。

2.考虑到中文编码问题，我们在"TXT版"文件夹中放置了所有代码的txt版本文件，可以用来看注释。

3.程序运行后会根据TestCase文件夹中的data.txt文件中的数据来生成放置的矩形位置文件result.txt，保存到TestCase文件夹中

4."运行程序.exe"必须与TestCase文件夹处于同一层级，不然将无法读取其中的data.txt文件信息。这一原则同样适用于重新生成的可执行文件

此外，在commontext.h文件中激活宏定义#define __PLACE_ORDER__可以在TestCase文件夹中生成Order.txt，里面记录放置矩形的顺序，如果在第一行添加“空间宽度 矩形总数"，该文件也可作为data.txt使用。
但要注意，每次生成Order.txt后，重新生成时，需要清除掉里面的内容，不然会默认在文件的末尾处添加新的信息。
